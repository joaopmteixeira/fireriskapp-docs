# Resumo do dia — 15 de maio de 2026

## O que foi feito hoje

- **Eliminado o tempo de espera de 45 segundos no login.** A base de dados estava alojada num serviço (Neon) que "adormecia" após 5 minutos sem atividade, fazendo com que o primeiro acesso depois de um período de inatividade demorasse quase um minuto. Mudámos para um serviço diferente (Supabase) que só adormece após uma semana — na prática, a base de dados está sempre "acordada" durante o uso normal. O login passou a demorar cerca de 1,5 segundos.

- **Resolvida uma falha crítica que fazia o servidor reiniciar em loop.** Ao ligar para a base de dados, o servidor estava a usar uma forma de gerir ligações que não é compatível com a forma como o servidor de produção (Render) cria processos paralelos. A solução foi ajustar a maneira como cada pedido acede à base de dados — o serviço de cloud continua a gerir as ligações de forma eficiente em segundo plano.

- **A nova versão do motor do servidor (FastAPI) entrou oficialmente em produção.** O trabalho de migração que estava a ser desenvolvido numa "branch" separada foi integrado na versão principal da aplicação. O servidor é agora mais moderno, mais bem organizado e mais fácil de manter.

- **Removidas pastas internas de ferramentas de trabalho do repositório público.** Ficheiros de configuração e notas de trabalho da ferramenta de desenvolvimento (Claude Code) estavam a ser publicados no GitHub desnecessariamente. Foram removidos do histórico público mas mantidos localmente.

- **Toda a documentação interna atualizada** para refletir o estado atual do projeto, incluindo os planos de trabalho e o registo de alterações.

## Estado do projeto

A aplicação está estável em produção com a nova arquitetura de servidor e a nova base de dados. O login é rápido e fiável. Toda a funcionalidade do modelo CHICHORRO 3.1 está operacional. O próximo foco será completar o modo escuro (dark mode) nas páginas ainda em falta, e possivelmente avançar com algumas das funcionalidades em lista de espera (como guardar avaliações de edifícios ou um gráfico de impacto das intervenções).
