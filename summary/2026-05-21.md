# Resumo do dia — 21 de maio de 2026

## O que foi feito hoje

- **A app passou a recusar arrancar em produção se faltar qualquer configuração crítica.** Antes, se uma chave secreta ou endereço de base de dados estivessem em falta, o servidor arrancava na mesma mas com comportamentos errados difíceis de diagnosticar. Agora dá erro imediato com mensagem clara.

- **Foram removidas credenciais de teste que estavam escondidas no código do site.** Um ficheiro antigo guardava nomes de utilizador e palavras-passe em texto claro dentro do código JavaScript público — qualquer pessoa que inspecionasse o site podia vê-los. O ficheiro foi eliminado por não estar a ser usado para nada.

- **O servidor ficou configurado para reconhecer que está atrás de um intermediário seguro.** O Render (serviço onde o backend está alojado) recebe as ligações em HTTPS mas passa-as ao servidor da app em HTTP interno. Documentámos o comando de arranque correto para que o servidor saiba que as ligações originais eram seguras.

- **Adicionada proteção contra configuração inválida dos cookies de sessão.** Se alguém definir um valor errado numa variável de ambiente dos cookies, o servidor agora rejeita o arranque com uma mensagem explicativa em vez de funcionar de forma imprevisível.

- **Foram criados resumos técnicos detalhados de tudo o que foi alterado.** Cada plano de audit tem agora um ficheiro de registo que documenta o que foi implementado, os problemas encontrados, e as decisões tomadas — útil para auditorias futuras.

- **A pasta de instalação local do Python foi excluída do repositório.** O VS Code tinha criado uma pasta com milhares de ficheiros de instalação do Python dentro do projeto; esses ficheiros não devem estar no repositório e foram excluídos permanentemente.

## Estado do projeto

O projeto está a meio de um ciclo de correções de segurança iniciado por uma auditoria automática (Codex). Quatro dos dezasseis planos de correção foram concluídos hoje — todos na branch `audit-fix`, que só será integrada no ramo principal quando o ciclo estiver completo. As funcionalidades da app em si não foram alteradas; todas as mudanças são de infraestrutura e segurança.
